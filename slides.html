Pain Points

痛点引入：我们日常移动端调试有多痛？

  • 配置折磨 (Proxy Hell)
    Charles / Fiddler 手动修改 IP、切换 Wi-Fi 重配，下班忘了关手机直接“断网”。
  • HTTPS 证书鸿沟 (SSL Trust)
    iOS 16+ / Android 7.0+ 安全机制严苛，安装、信任证书步骤深陷迷宫。
  • 手机造数极其低效
    真机屏幕太小，改个超长 JSON 或模拟各种极端报错结构极其痛苦。
  • 多人规则干扰冲突
    常规服务器 Mock 是全局共用的，规则容易被队友覆盖、冲突，且不支持离线使用。

Architecture

工作原理：免代理直连与智能路由分流

  • 客户端一键直连：
    手机扫控制台二维码 ➔ 在 App 内自动将本地 Mock 服务地址绑定为请求前缀。
  • 直连重定向拦截 (URL Rewrite)：
    https://api.com/user 重定向为 http://<PC_IP>:8099/mock/api.com/user.
  • 匹配路由分流：
    命中 Mock 规则，直接返回本地 JSON。未命中规则，扮演透明代理自动向真实网关转发并返回真实数据。
  • 流量实时追踪：
    服务端通过 httpx 异步拉取响应并在 Web 控制台记录时序、耗时及报文信息。
Integration

客户端零侵入式集成：Swift & Kotlin 核心实现

  • 极简拦截适配：
    在 Moya / Alamofire (iOS) 或 OkHttp (Android) 底层添加一个 Interceptor。
  • 一份代码，全局收益：
    只在 Mock 开关打开且扫码连接配置存在时，克隆并重定向 `URLRequest`。
  • 安全透明：
    不影响生产环境，对原本的核心业务代码零侵入、零污染。

💡 技术小贴士： 拦截器会自动在请求头追加 `X-Original-URL` 头部，以便服务器精确匹配。

LittleCatInterceptor.swift 
struct LittleCatInterceptor: RequestInterceptor {
    func adapt(_ urlRequest: URLRequest, ...) -> URLRequest {
        guard MockManager.shared.isEnabled,
              let mockServer = MockManager.shared.serverIp else {
            return urlRequest
        }
        
        var request = urlRequest
        if let originalUrl = urlRequest.url {
            // 重写 URL 前缀 (仅保留 Path)，直连小猫 PC 服务器
            request.url = rewrite(path: originalUrl.path, to: mockServer)
            request.setValue(originalUrl.absoluteString, forHTTPHeaderField: "X-Original-URL")
            request.setValue(originalUrl.host ?? "", forHTTPHeaderField: "X-Original-Host")
        }
        return request
    }
}
AI Innovation

AI 智能 Mock 与稳定性混沌变异测试

  • AI 业务范式 Mock：
    深度整合 DeepSeek / Claude，根据 API Path 与入参，自动生成逻辑清晰、结构拟真的 JSON，告别手写假数据。
  • 流式 SSE 输出体验：
    整合 SSE 流式渲染，使长文本大模型响应极其流畅地推送到终端。
  • AI 混沌变异 (Chaos Test)：
    大模型自适应注入“破坏性异常”：
    • 字段缺失 (随机清空必填项为 null)
    • 数据类型冲突 (数字转为字符串 "100")
    • 极值与边界溢出 (超长特殊乱码等)
Web Panel

极客风 Web 大屏控制台与高级特性

  • 极具现代感的双色皮肤：
    光感 Light 主题与暗黑极客 Dark 皮肤一键切换，舒适护眼。
  • 可视化 JSON 规则树编辑器：

    请求列表双向瞬时检索，支持右键一键复制标准 cURL 调试命令。
DevOps

单文件极简分发与物理外置持久化

  • 0 Python 环境依赖分发：
    利用 PyInstaller 与专属打包脚本 package.py，将 FastAPI 运行时、网络工具及前端资源一键打成独立 `.exe` / `.app`。
  • 静态资源内含解压 (MEIPASS)：
    Web UI 模板以纯二进制形态封包，运行时在内存中极其流畅地解压释放，极速无感。
  • 物理路径持久化 (Anti-Loss)：
    软件运行时自动识别运行环境，将用户的 Mock 规则数据持久化保存在物理外置目录下。
  • 无痛升级升级不丢规则：
    完美解决了重装、覆盖安装软件后历史造数规则彻底丢失的窘境。
